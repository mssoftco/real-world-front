## Getting started

### [Live Demo](https://mssoftco.gitlab.io/arvan-cloud-front/)

or

1- install packages:

```bash
npm install
```

2- run the development local:

```bash
npm run dev
```

3- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
